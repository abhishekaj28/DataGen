# ◈ DataGen Framework

> **Automated Synthetic Dataset Generation for ML Fine-Tuning**

![Python](https://img.shields.io/badge/Python-3.14-blue) ![FastAPI](https://img.shields.io/badge/FastAPI-0.100-green) ![React](https://img.shields.io/badge/React-18-cyan) ![TypeScript](https://img.shields.io/badge/TypeScript-5.0-blue)

## 🔍 The Problem
- Real-world training data is scarce, expensive & legally restricted
- Manual labeling takes weeks and introduces human bias
- ML teams waste months collecting data instead of building models

## ✨ Solution
DataGen uses LLMs to automatically generate, validate, and export privacy-safe synthetic datasets — in minutes, not weeks.

## 🧠 How It Works
```
Task Config → LLM Generation → Auto Validation → Bias Analysis → Export
```

## 🎯 Supported Task Types
| Task | Description |
|------|-------------|
| Classification | Text → Label |
| Question Answering | QA Pairs |
| Summarization | Long → Short |
| NER | Entity Extraction |
| Intent Detection | Utterance → Intent |

## 🏗️ Architecture
```
React Frontend (TypeScript)
        ↕  REST API
FastAPI Backend (Python)
        ↕  API Calls
Claude / OpenAI (LLM)
```

## 🚀 Quick Start

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

- Backend: http://localhost:8000
- Frontend: http://localhost:8080
- API Docs: http://localhost:8000/docs

## 💡 Key Features
- **5 Task Types** — Classification, QA, Summarization, NER, Intent Detection
- **Auto Validation** — Every sample scored for quality
- **Edge Cases** — Adversarial examples generated by default
- **Privacy Safe** — No real user data needed
- **Export Ready** — JSON + CSV download

## 🛠️ Tech Stack
- **Frontend:** React, TypeScript, Tailwind CSS, Vite
- **Backend:** FastAPI, Python, Pydantic
- **AI:** Anthropic Claude / OpenAI GPT
