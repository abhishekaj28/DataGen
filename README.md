## 🔍 The Problem

* Real-world training data is scarce, expensive & legally restricted
* Manual labeling takes weeks and introduces human bias
* ML teams waste months collecting data instead of building models

---

## ✨ Solution

DataGen uses LLMs to automatically generate, validate, and export privacy-safe synthetic datasets — in minutes, not weeks.

---

## 🧠 How It Works

```
User Input  
   ↓  
LLM Data Generation  
   ↓  
Quality Validation  
   ↓  
Bias & Edge Case Analysis  
   ↓  
Export (JSON / CSV)
```

## 🎯 Supported Task Types

| Task               | Description        |
| ------------------ | ------------------ |
| Classification     | Text → Label       |
| Question Answering | QA Pairs           |
| Summarization      | Long → Short       |
| NER                | Entity Extraction  |
| Intent Detection   | Utterance → Intent |

## 🔌 API Endpoints

| Method | Endpoint  | Description      |
| ------ | --------- | ---------------- |
| POST   | /generate | Generate dataset |
| POST   | /validate | Validate dataset |
| GET    | /export   | Export dataset   |

## 🚀 Quick Start

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

* Backend: http://localhost:8000
* Frontend: http://localhost:8080
* API Docs: http://localhost:8000/docs
